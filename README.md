# Scuderia_Smartcar_Mission_Doublenow

MISSION DOUBLENOW!

东南大学第十四届智能车校赛双车接力组，拟参加第十五届全国大学生智能车竞赛。

## 项目介绍

目前拟使用 *电磁F车* 作为送球车，*某摄像头四轮车* 作为接球车。校一要求只要做出送球车部分即可。
