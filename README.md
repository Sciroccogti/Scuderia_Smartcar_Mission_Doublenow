## 项目介绍

使用 *电磁F车* 作为送球车，*摄像头C车* 作为接球车，使用电磁铁传球，送球车在被碰撞（编码器读值变化）后发车。

[配套的摄像头C车代码](https://github.com/zhongmou3/smartcar)
